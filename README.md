Guess Number Game

One of my first Python projects, created as part of my Python learning journey.

In this game, the player chooses the range of numbers. The program then generates a random number within that range, and the player has three chances to guess it correctly.

If the player loses, they have the option to start a new game and try again.

Features

- User-defined number range
- Random number generation
- Three guessing attempts
- Higher / Lower hints
- Option to play again after losing

Built With

- Python 3

Learning Goals

This project helped me practice:

- Variables
- User input
- Conditional statements
- Loops
- Functions
- Random number generation
